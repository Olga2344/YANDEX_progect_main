# YANDEX_progect_main
 Проекты основого курса аналитики
В репозитории собраны три учебных аналитических проекта по продуктовой и маркетинговой аналитике.  
Каждый проект оформлен в виде Jupyter Notebook и имеет свой `README.md` с подробным описанием задачи, данных, стека и выводов.

### Состав репозитория
- `Проект пятого модуля 3 V.ipynb` — анализ юнит‑экономики и источников привлечения приложения **Procrastinate Pro+**:
    - расчёт DAU, MAU, stickiness;
    - оценка CAC, LTV, ROI в горизонте 28 дней;
    - анализ окупаемости рекламных каналов и рекомендации маркетингу.
  - `README.md` — описание постановки задачи, данных, стека и выводов по проекту.

- `Итоговое 6 модуль V.ipynb` — анализ воронок и A/B‑тест нового онбординга инвестиционного сервиса **SollmaFin**:
    - воронки событий и узкие места;
    - влияние риска первого актива на конверсию во второй депозит;
    - оценка A/B‑теста с помощью χ², t‑теста и бутстрапа перцентилей.
  - `README.md` — подробное описание эксперимента, метрик и результатов.

- `Итоговое 7 модуль V.ipynb` — аналитика маркетплейса и A/B‑тест бонуса для пользователей **TikTok**:
    - обзор ключевых продуктовых и финансовых метрик (ARPU, AOV, CAC, ROI, payback);
    - когортный и сегментационный анализ (каналы, регионы, сегменты покупателей);
    - A/B‑тест бонуса на первую покупку для TikTok‑пользователей и оценка эффекта.
  - `README.md` — описание данных, метрик юнит‑экономики и результатов эксперимента.
